Запуск:
1) java -jar mvk-1.0.jar  обновление данных через API
2) java -jar mvk-1.0.jar -f обновление данных при попомщи самого нового файла в папке temp
3) java -jar mvk-1.0.jar -f "c:\folder\file.xml" обновление при помощи указанного файла


Самое простое положить файл в папку temp в  каталоге с приложением и запустить: 
java -jar mvk-1.0.jar -f
